# Netflix-powerbi-dashboard

An interactive Power BI dashboard analyzing Netflix content distribution across
countries, ratings, and release trends.

# Project Overview
This project explores the Netflix titles dataset to uncover insights about content distribution
by country, rating categories, content type, and release trends over time. The dashboard
was built using Power BI Desktop with custom DAX measures and a Netflix-branded dark
theme.

# Key Insights
8,796 total titles available on Netflix
748 countries represented in the dataset
United States leads with 3,641 titles
India is second with 972 titles
Average movie duration is 99.59 minutes
Netflix content grew dramatically after 2015
Movies outnumber TV Shows in the catalog
TV-MA is the most common content rating

# Tools Used
Power BI Desktop Dashboard creation & visualization
DAX Custom measures (Avg Duration, Content Count)
SQL Data exploration and cleaning
GitHub Version control and portfolio hosting

# Dataset
Source: Netflix Titles Dataset - Kaggle
Records: 8,796 titles
Fields: Title, Type, Country, Rating, Duration, Release Year, Category

# Dashboard Features
Total Content KPI — Total number of Netflix titles
Total Countries KPI — Number of countries with Netflix content
Avg Movie Duration — Average runtime for movies in minutes
Content Type Donut Chart — Movies vs TV Shows breakdown
Rating By Country Bar Chart — Content count per country
Content By Rating Chart — Distribution across rating categories
Shows By Release Year Line Chart — Content growth over time
Interactive Slicers — Filter by Country, Category, and Release Date

# How to View
1. Download the .pbix file from this repository
2. Open with Power BI Desktop (free)
3. Explore the interactive dashboard
